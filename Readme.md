## Installation:
```bash
git clone https://github.com/Lucksi/Darkus
cd Darkus
sudo apt-get update
sudo chmod +x Installer.sh
sudo bash Installer.sh
```

## Execution:
```bash
cd Darkus
python3 Main.py
```
